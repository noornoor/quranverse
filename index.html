Easily embed a verse from Qur'aan


Here's how to easily embed a verse from the Qur'aan on any website.

Paste below html snippet into your webpage: 
<script chapter="15" verse="9" 
   src="//noornoor.github.io/quranverse/js/quran.js">
</script>




An example with english translation included. Just add another parameter called trans like so:

<script chapter="15" verse="9" trans="#translation"
   src="//noornoor.github.io/quranverse/js/quran.js">
</script><div id='translation'></div>




An example showing wordbyword. Just add a parameter called qurantype like so:

<script chapter="15" verse="9" qurantype="wordbyword"
   src="//noornoor.github.io/quranverse/js/quran.js">
</script>




Another example showing a different qurantype - grammar:

<script chapter="15" verse="9" qurantype="grammar"
   src="//noornoor.github.io/quranverse/js/quran.js">
</script>

   src="http://noornoor.github.io/quranverse/js/quran.js">
</script>
